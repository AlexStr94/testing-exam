# Wikipedia Web UI Tests

## Описание проекта

Это тестовый проект для автоматизированного тестирования публичного сайта Wikipedia (https://www.wikipedia.org) с использованием Java, Selenium WebDriver и TestNG.  
Проект реализует четыре ключевых сценария проверки пользовательского интерфейса:

1. Открытие главной страницы и проверка заголовка.
2. Поиск статьи по ключевому слову и проверка заголовка открытой статьи.
3. Переход по первой ссылке внутри статьи и проверка перехода.
4. Смена языка интерфейса и проверка URL выбранной версии.

Проект демонстрирует использование **Page Object Model (POM)**, явных ожиданий и стабильного запуска тестов на локальном компьютере.

## Настройка проекта

### 1. Java и Maven

Убедитесь, что установлены:

```bash
java -version    # Java 11+
mvn -version     # Maven 3.8+
```

### Запуск тестов

```
mvn test
```

## Сценарии тестирования

1. Открытие главной страницы
Проверка, что заголовок страницы содержит "Wikipedia".

2. Поиск статьи
Ввод ключевого слова в поле поиска и проверка заголовка открытой статьи.

3. Переход по ссылке статьи
Клик по первой ссылке в статье и проверка, что страница изменилась.

4. Смена языка интерфейса
Переход на английскую версию сайта и проверка URL.
